Keyboard Shortcuts

Moving the editor cursor

  • Up - negative Y
  • Down - positive Y
  • Left - negative X
  • Right - positive X
  • Page Up - negative Z
  • Page Down - positive Z

Changing the cursor direction

  • Tab - change the active plane
  • Alt+Up - negative Y
  • Alt+Down - positive Y
  • Alt+Left - negative X
  • Alt+Right - positive X

Viewing extra dimensions

  • Ctrl+Up - show three more dimensions
  • Ctrl+Down - view the selected set of three dimensions

Altering the view

  • Mousewheel scroll - zoom in/out
  • Ctrl+Middle mouse - rotate
  • Ctrl+Left mouse - move
